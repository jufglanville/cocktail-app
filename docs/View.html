Class: View

View

new View()

views/View.js, line 11

Represents a view made up of multiple components.

The View class acts as a blueprint for every object that extends it, providing ways to dynamically build the HTML content via the use of components. Children classes only have to populate the 'components' array using valid classes that extend 'Component', so that the render method builds them sequentially.
